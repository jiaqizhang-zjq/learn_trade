## 📋 文档管理规范

### 文件命名规范
- **学习计划类**: `XX_描述.md`（如：`01_学习路线图.md`）
- **书籍笔记类**: `XX_书名.md`（如：`01_富爸爸穷爸爸.md`）
- **知识体系类**: `XX_知识领域.md`（如：`01_基础知识.md`）
- **日期类**: `YYYY-MM-DD.md`（如：`2026-04-06.md`）

### 文件存放位置
| 文档类型 | 存放目录 | 示例 |
|---------|---------|------|
| 学习计划 | `01_学习计划/` | 学习路线图.md |
| 阅读笔记 | `02_阅读笔记/书籍笔记/` | 01_富爸爸穷爸爸.md |
| 知识体系 | `03_知识体系/` | 股票投资知识体系图谱.md |
| 实践记录 | `04_实践记录/模拟交易/` | 2026-04-06.md |
| 资源汇总 | `05_资源汇总/` | 推荐书籍.md |
| 心得体会 | `06_心得体会/月度总结/` | 2026-04-总结.md |
| AI配置 | `.trae/` | project_rules.md |

### 禁止行为
- ❌ 根目录（除README.md外）不放任何文件
- ❌ 不随意创建新目录
- ❌ 文件命名不使用中文特殊符号
- ❌ 不将笔记文件放在学习计划目录

## 💡 学习原则

1. **循序渐进**: 按照阶段顺序学习，打好基础
2. **理论结合实践**: 每学完一个知识点就尝试应用
3. **持续记录**: 养成做笔记的习惯，便于复习
4. **定期复盘**: 每周、每月总结学习成果
5. **保持耐心**: 投资是一辈子的修行，不要急于求成
