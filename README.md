## 📝 使用指南

### 如何记录笔记

1. **书籍笔记**：复制模板，按章节记录要点
2. **文章笔记**：记录核心观点和个人思考
3. **研报笔记**：整理关键数据和结论

### 如何制定计划

1. **每周计划**：使用模板制定下周学习目标
2. **月度总结**：回顾本月学习成果
3. **调整优化**：根据实际情况调整计划

---

## 🎯 学习原则

1. **循序渐进**：按照阶段顺序学习，打好基础
2. **理论结合实践**：每学完一个知识点就尝试应用
3. **持续记录**：养成做笔记的习惯，便于复习
4. **定期复盘**：每周、每月总结学习成果
5. **保持耐心**：投资是一辈子的修行，不要急于求成
