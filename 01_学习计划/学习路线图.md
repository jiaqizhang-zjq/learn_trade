# 股票学习路线图

## 一、基础知识阶段（1-2个月）

### 1. 股市基础概念
- [ ] 了解股票的基本概念和分类
- [ ] 学习股市术语（多头、空头、涨停、跌停等）
- [ ] 理解交易规则（T+1、涨跌幅限制等）
- [ ] 熟悉交易流程和费用

### 2. 财务基础知识
- [ ] 学习财务报表（资产负债表、利润表、现金流量表）
- [ ] 掌握财务指标（PE、PB、ROE、ROA等）
- [ ] 理解财务分析方法

### 3. 宏观经济基础
- [ ] 了解宏观经济指标（GDP、CPI、PPI等）
- [ ] 学习货币政策与财政政策
- [ ] 理解经济周期

## 二、技术分析阶段（2-3个月）

### 1. K线基础
- [ ] 学习K线图的基本构成
- [ ] 掌握常见K线形态
- [ ] 理解K线组合形态

### 2. 技术指标
- [ ] 移动平均线（MA、EMA）
- [ ] 相对强弱指标（RSI）
- [ ] 随机指标（KDJ）
- [ ] 布林带（BOLL）
- [ ] MACD指标

### 3. 图表形态
- [ ] 头肩顶/头肩底
- [ ] 双顶/双底
- [ ] 三角形形态
- [ ] 旗形、楔形

### 4. 成交量分析
- [ ] 量价关系
- [ ] 成交量指标
- [ ] 筹码分布

## 三、基本面分析阶段（2-3个月）

### 1. 行业分析
- [ ] 行业分类与特征
- [ ] 行业生命周期
- [ ] 行业竞争格局
- [ ] 行业政策分析

### 2. 公司分析
- [ ] 商业模式分析
- [ ] 竞争优势分析
- [ ] 管理层分析
- [ ] 财务深度分析

### 3. 估值方法
- [ ] 相对估值法（PE、PB、PS等）
- [ ] 绝对估值法（DCF、DDM等）
- [ ] 估值实战应用

## 四、交易策略阶段（3-4个月）

### 1. 交易系统构建
- [ ] 选股策略
- [ ] 买卖时机
- [ ] 仓位管理
- [ ] 止损止盈

### 2. 常见策略学习
- [ ] 价值投资策略
- [ ] 成长投资策略
- [ ] 趋势跟踪策略
- [ ] 波段操作策略

### 3. 风险管理
- [ ] 资金管理原则
- [ ] 风险控制方法
- [ ] 心理建设

## 五、实践与提升阶段（持续）

### 1. 模拟交易
- [ ] 开通模拟账户
- [ ] 实践交易策略
- [ ] 记录交易过程

### 2. 实盘操作
- [ ] 小资金试水
- [ ] 逐步加仓
- [ ] 定期复盘

### 3. 持续学习
- [ ] 阅读经典书籍
- [ ] 关注市场动态
- [ ] 总结经验教训

## 学习建议

1. **循序渐进**：按照路线图逐步学习，不要跳跃
2. **理论结合实践**：学习理论知识的同时进行模拟交易
3. **记录总结**：养成做笔记和复盘的习惯
4. **持续学习**：股市变化万千，需要不断学习和适应
5. **风险意识**：始终牢记风险控制，保护本金

## 时间安排建议

- **工作日**：每天1-2小时学习
- **周末**：每周4-6小时深度学习
- **每月**：进行一次总结和复盘
