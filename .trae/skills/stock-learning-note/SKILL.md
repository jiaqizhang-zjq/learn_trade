# 股票学习笔记记录 Skill

## 何时调用

在以下情况下调用此 Skill：
- 用户说"我今天读了..."
- 用户分享阅读体会或笔记
- 用户说"记录学习进度"
- 用户提到具体章节内容
- 用户说"继续记录阅读笔记"

## 功能说明

这个 Skill 用于自动化股票学习笔记记录流程：

1. **读取当前状态** - 从 `当前阅读状态.md` 获取阅读进度
2. **整理笔记** - 将用户的体会整理成结构化格式
3. **更新进度** - 更新当前阅读状态和书籍笔记
4. **提交保存** - Git 提交到 GitHub

## 使用方法

### 标准流程

```
用户: "我今天读了《富爸爸穷爸爸》第3章，关于资产和负债的..."

AI:
1. 读取 当前阅读状态.md
2. 确认当前书籍和章节
3. 整理用户分享的笔记
4. 更新 当前阅读状态.md 进度
5. 更新 02_阅读笔记/书籍笔记/01_富爸爸穷爸爸.md
6. Git 提交
7. 反馈更新结果
```

### 关键文件

- **当前阅读状态**: `当前阅读状态.md`
- **书籍笔记**: `02_阅读笔记/书籍笔记/XX_书名.md`
- **笔记模板**: `02_阅读笔记/笔记整理模板.md`

### 笔记格式

整理后的笔记包含：
- 日期和阅读时长
- 章节核心内容概括
- 用户的重要体会
- 经典语录摘录
- 实践任务清单
- 下一步计划

## 注意事项

1. **自动识别书籍** - 从用户描述中识别书名和章节
2. **保持连续性** - 基于之前的内容继续记录
3. **结构化整理** - 使用统一的笔记格式
4. **及时提交** - 每次记录后提交到 GitHub

## 输出示例

**用户输入**:
"我今天读了第3章，关于资产和负债的区别。最大的收获是明白了房子不一定是资产，如果它每个月让你花钱就是负债。"

**AI 整理后**:
```markdown
### 第3章学习记录 - 2026-04-06

**核心观点**:
- 资产是把钱放进你口袋的东西
- 负债是把钱从你口袋拿走的东西
- 房子不一定是资产

**我的体会**:
- 颠覆了传统观念
- 要用现金流思维看待资产

**待实践**:
- [ ] 列出自己的资产和负债清单
```
